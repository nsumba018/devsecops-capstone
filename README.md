# DevSecOps Capstone Lab

## Overview
This project demonstrates a complete DevSecOps deployment pipeline using:

- AWS EC2
- Terraform
- Ansible
- Docker
- Docker Compose
- NGINX
- GitLab CI/CD

## Architecture

Frontend (NGINX)
↓
Backend (Node.js Express)
↓
PostgreSQL

## Deployment Steps

1. Provision infrastructure with Terraform
2. Configure server with Ansible
3. Deploy containers using Docker Compose
4. Access application through EC2 public IP
